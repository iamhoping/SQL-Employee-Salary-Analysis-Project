# SQL-Employee-Salary-Analysis-Project
This project contains SQL practice queries focused on employee salary analysis, department insights, and relational database operations.  The queries analyze employee and department data using joins, aggregate functions, grouping, filtering, sorting, and subqueries.

## Description
This project contains SQL practice queries focused on employee salary analysis and department insights.

## SQL Concepts Used
-SELECT
-WHERE
-ORDER BY
-GROUP BY
-JOIN
-INNER JOIN
-RIGHT JOIN
-Aggregate Functions:
-AVG()
-COUNT()
-MAX()
-ROUND()
-Subqueries
- cte
- temp table
-Aliases

## Example Queries
-Average salary by age group
-Highest-paid employee per department
-Employees earning above department average
-Department salary comparison
-Employee filtering and sorting
-Occupation-based employee analysis

## Sample Analysis Included
-Group employees by age and calculate average salary
-Compare department salaries against company average
-Identify top earners in each department
-Sort employees alphabetically
-Analyze occupations and salary distributions

## Purpose
Created to practice relational database querying and SQL analysis skills.
